🐾 Pet Shop Website – Front-End Project (Group 21)

เว็บไซต์ร้านขายอุปกรณ์และอาหารสัตว์เลี้ยง พัฒนาในรูปแบบ Front-End (Static Website)
จัดทำเพื่อฝึกการออกแบบและพัฒนาเว็บไซต์ด้วย HTML และ CSS

📌 รายละเอียดโปรเจกต์

เว็บไซต์นี้เป็นเว็บร้านขายสินค้าเกี่ยวกับสัตว์เลี้ยง เช่น:

🐶 อาหารสุนัข

🐱 อาหารแมว

🧸 ของเล่นสัตว์เลี้ยง

🧴 ผลิตภัณฑ์ดูแลสัตว์

🛒 อุปกรณ์สัตว์เลี้ยง

📄 หน้ารายละเอียดสินค้า (Product Details)

โครงสร้างเว็บไซต์แบ่งหมวดหมู่ชัดเจน และมีหน้าแสดงรายละเอียดสินค้าแยกแต่ละรายการ

🛠 เทคโนโลยีที่ใช้

HTML5

CSS3

javaScript (ไม่ได้สร้างไฟล์แยก)

โครงสร้างแบบ Static Website

(บางหน้ามีการใช้ Tailwind CSS ผ่าน CDN)


🔎 คำอธิบายโฟลเดอร์สำคัญ

Home-page/ → หน้าแรกของเว็บไซต์

category/ → หน้าหมวดหมู่สินค้า

dog_food/ → หน้าอาหารสุนัข

cat_food/ → หน้าอาหารแมว

Pet toys/ → ของเล่นสัตว์เลี้ยง

Pet supplies/ → อุปกรณ์สัตว์เลี้ยง

Pet care products/ → ผลิตภัณฑ์ดูแลสัตว์

Product details/ → หน้ารายละเอียดสินค้าแต่ละชิ้น

Pictures/ → รูปภาพสินค้าและรูปประกอบ
